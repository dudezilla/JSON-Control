eventLinkage(state_change)

Called during runtime after construction - eventLinkage can be called during an "Event" This is an entry point for the event handler. This method is used to pass a message from a child control's event-hanlder to the parent control.

The nature of the message is arbitrary and is interpreted by the parent control.
